SCREEN 03 / 07
クイズ画面
チャット後に挿入されるSNSリテラシー/マーケティングのミニクイズ。全ルート共通。3択で正解/不正解のフィードバックを表示する。
クイズ本体
Q. SNS リテラシー
フォロワー数よりも重要視されるSNS指標として正しいものはどれですか？
A
フォロワー数の絶対値
B
エンゲージメント率（いいね・コメント数 / リーチ数）
C
投稿の文字数
正解です！エンゲージメント率は「届いた人の何割が反応したか」を示します。バズっていても反応率が低い投稿は、実はターゲットに刺さっていない可能性があります。
仕様・注釈
出題タイミング：各ルートのチャット終了後、素材確認画面の前に挿入。

問題数：ルートごとに異なる（みどり：2問、朔/誠司：3問、花蓮：4問）。

スコア影響：正解でメッセージ表示のみ。ゲームスコアへの直接影響なし（学習性演出）。

プログレスバー：上部の5ドットが現在位置を示す（done: midori / active: blue）。

フィードバック：選択後に下部に展開。解説テキスト + 「次へ」ボタン。
インタラクション：クリックで即座にフィードバックが表示される（アニメーション: slideDown 200ms）。正解は緑、不正解は赤でハイライト。次の問題へ自動スクロール。
選択肢数: 3択（固定）
問題テキスト: 最大 120文字
解説テキスト: 最大 200文字
画面: appWindow 内の quiz エリア

エンドロールへ
エンディング分岐条件（全8種）：
🌿 自立エンド: selfScore ≥ 65（みどりルート）
🌱 継続エンド: selfScore < 65 かつ buzzScore < 65（みどりルート デフォルト）
🌙 到達エンド: selfScore ≥ 60 かつ kShinonoya=true（朔ルート）
☕ 再訪エンド: buzzScore < 65 かつ selfScore < 65（誠司ルート デフォルト）
🌸 既読エンド: selfScore ≥ 65 かつ flashbackPhase ≥ 3（花蓮ルート）
🔥 ズレエンド: buzzScore ≥ 65 かつ selfScore < 50（全ルート）
🎭 贖罪エンド: kidoku 経由 / egoScore > 0（花蓮ルート後）
🖤 沈黙エンド: kidoku 経由 / egoScore ≤ 0（花蓮ルート後）
※ zange / chinmoku は kidoku の lastChoice を経由して到達。直接到達するパスなし。
※ ルートによって条件が異なる（詳細は機能設計書 §6 参照）。
