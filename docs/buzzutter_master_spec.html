元フリーライター → SNS運用代行
「言葉は刃になる。それを知っているから、今は他人の言葉の翻訳者として生きている。」
過去
3年前、バズ目的の告発記事を書いた。対象者（篠宮カレン）がSNSを削除し消息不明に。以来ライター業を断ち、SNS運用代行として生きている。
なぜこの仕事か
「バズらせること」で誰かを傷つけた贖罪として「その人らしい言葉を守る仕事」を選んだ。バズを追わないのは信念であり、トラウマ。
自我スコア（egoScore）
選択肢の累積で決まる。「感情を出す」→ +1 / 「仕事に徹する」→ -1。正：贖罪エンド、負：沈黙エンド。
テキスト上の注意：主人公の台詞は「賢人化」させない。「全員じゃなくていい。1人に刺さればそれが答えです」のような格好よすぎる台詞NG。迷い・不確かさ・弱さを残すこと。
