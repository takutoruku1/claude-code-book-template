# fun_review_tension — テンションカーブ評価

**評価日:** 2026-05-11
**評価者:** テンションカーブ担当（面白さ保証チーム②）
**総合グレード:** C

---

## サマリー

4ルートを通じて、いずれもテンションの立ち上がりが緩慢で、ピーク前の「溜め」設計が不十分という共通の弱点がある。みどりルートは素材フェーズ前後でテンションが完全に平坦化する。朔ルートはフラッシュバックとモノローグが唯一の引っ張り手となっており、ピーク直前の溜めシーンが1ステップしかない。誠司ルートはピーク自体がスコア6止まりという「ピーク不足」状態で、唯一の緊張源が湊のモノローグ（内心のみ）に集中している。花蓮ルートのみ構造的なピーク（スコア9〜10）を持ち、謎収束設計も機能しているが、序盤の素材フェーズ付近に「低平坦ゾーン」がある。全ルートを通じて、素材選択フェーズ（trigger: showMaterial）がテンションの谷になっており、パターンB（素材選択平坦化）が全ルートに波及している点が最優先の改善課題である。

- ただし karen_system/trigger（スコア3）の急落後、showPost（スコア5）→ karen_pre_mystery（スコア7）と回復するまでに2ステップ必要で、この区間が「溜めの充電コスト」として機能を弱めている

**問題点:**
- **[重要度: 高] 素材フェーズ（karen_system / trigger）のスコア3急落:** karen_question_reply（スコア8）の緊張感が素材フェーズで完全にリセットされる。「昔ライターだったか？」という核心的な問いの直後に事務作業が来る構造は、緊張感を殺すタイミングとして最悪。このルートで最も大きなテンション破綻点。
- **[重要度: 高] karen_endパスの余韻設計がない:** karen_mystery_choiceで「（何も言わない）」を選んだ場合、karen_end（空テキスト終端マーカー）に到達し即終了する。この沈黙パスには余韻がなく、スコア9→終了という急落になっている。
- **[重要度: 中] 花蓮ルート序盤（スコア4〜5が3連続）の引力が弱い:** karen_helloからhello3まで、キャラクターの奇妙さや謎感が薄い。「記録として。形として。残したい。」という発言（hello3）は設計上スコア5相当だが、これは花蓮の正体を知っているプレイヤーにしか機能しない。初見ではただの静かな依頼に見える。
- **[重要度: 中] karen_mystery_choiceがcondition付き（mysteryClues >= 2）のため、条件不成立時は謎収束シーン自体が存在しない:** 条件を満たさないプレイヤーにはkaren_endに直行し、スコア10のシーンに到達できない。このルートのピーク自体が条件付きアクセスになっている点は、体験の格差として問題がある。

## 全ルート共通の問題点（優先度順）

### 1. [最優先] 素材フェーズ（showMaterial trigger）が全ルートでテンション谷になっている
**該当箇所:** midori_system/trigger（スコア2）、saku_trigger（スコア3）、seiji_system/trigger（スコア2）、karen_system/trigger（スコア3）

全ルートで素材フェーズ直前〜中に依頼主の声や湊のモノローグが存在しないため、テンションが2〜3まで急落する。仕様書「パターンB」で指摘されている「素材選択平坦化」がそのまま全ルートに発生している。

**改善方向:**
- 素材フェーズ開始直前に依頼主から1行（期待・不安を含む短い一言）を追加する
- 素材フェーズ中に湊の内心モノローグ（「これを使うか…」「バレるか」程度）をtrigger内に仕込む
- 素材フェーズ完了直後のresumeAt先に、依頼主の反応または湊の内心を必ず1ステップ追加する

---

### 2. [優先] 誠司ルートのピーク不足（最高スコア6）
**該当箇所:** 誠司ルート全体

全ルート中唯一、スコア7以上に到達しない。「バレたら終わり」の具体的な緊張シーンが設計されていないことが根本原因。

**改善方向:**
- 投稿後のリアクションフェーズで「ゆきわりそうの関係者（または花蓮）から何らかの反応が来る」演出を追加し、スコア8相当の緊張シーンを1箇所設ける
- または誠司との会話の中で「以前来ていた若い女性がSNSで書かれた記事を読んでいた可能性」をほのめかすことで、湊の代行と過去が交差する緊張ポイントを作る

---

### 3. [優先] 朔ルート・花蓮ルートの素材フェーズ直前が感情的高点であるためテンション落差が最大になっている
**該当箇所:** saku_send（スコア6）→saku_trigger（スコア3）、karen_question_reply（スコア8）→karen_trigger（スコア3）

素材フェーズの急落はどのルートでも発生するが、朔ルートと花蓮ルートでは直前が感情的高点（スコア6〜8）のため落差が最も大きく、体験の断絶感が著しい。

**改善方向:**
- 素材フェーズ直前に意図的にテンションを1〜2段階下げる「ブレスステップ」を挿入し、落差を緩和する（例：「少し考えた。素材を確認しよう」等の短いモノローグ）
- または素材フェーズ自体に緊張要素を埋め込む（上記1の対処と連動）

---

### 4. [推奨] みどりルートの序盤低平坦ゾーン（8ステップ）の解消
**該当箇所:** midori_hello〜midori_plant_reply（スコア3〜4が8ステップ連続）

チュートリアルとして緩やかな設計は合理的だが、8ステップ連続の低平坦は仕様書の基準（3連続でフラグ）を大幅に超えている。

**改善方向:**
- midori_plant_detailかmidori_photo_talkのいずれかに「嫌な予感」要素を1行追加する（例：「この人の日常を代わりに書く、それが正しいのか——一瞬だけ考えた」）
- みどりの写真送付に「え、これ本当に私が撮ったってバレても大丈夫ですか？」のような一言を追加し、代行の構造的リスクをプレイヤーに早期に意識させる

---

### 5. [推奨] 花蓮ルートのkaren_endパス（沈黙選択肢）の余韻設計追加
**該当箇所:** karen_mystery_choice → karen_end（空テキスト終端）

「何も言わない」を選んだプレイヤーに対してスコア9から即終了は余韻が皆無。これはエンド分岐の中でも特に感情的余韻を必要とするシーン（言えなかった、の沈黙）であるはずだが、現状は設計の空白地帯になっている。

**改善方向:**
- karen_endに湊のモノローグ（2〜3行程度）を追加し、沈黙を選んだことの重さを描写する
- これは沈黙エンド（chinmoku）方向への橋渡しとして機能させることができる
