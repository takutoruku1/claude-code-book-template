自立エンド — みどり / selfScore 65以上
継続エンド — みどり / 両方65未満
到達エンド — 朔 / self60以上 + kShinonoya
再訪エンド — 誠司 / 両方65未満
既読エンド — 花蓮 / self65以上 + phase3以上
ズレエンド — 全ルート / buzz65以上 self50未満
贖罪エンド — kidoku後 / egoScore 0より大
沈黙エンド — kidoku後 / egoScore 0以下
贖罪エンド / 沈黙エンドは既読エンド（kidoku）を経由した場合のみ到達可能。直接到達するパスは存在しない。
ズレエンドはいずれのルートでも、buzzScore が高く selfScore が低い場合に発生する。
